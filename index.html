     * [(n)<t{a:x:y|b|c|...}h><s>][remark]
     *
     * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * *
     *
     *  <n>
     *  Loop count, repeat n
     *
     * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * *
     *
     *  <t>
     *  Template, template for use in a processor
     *
     * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * *
     *
     *  <a:x:y|b|c|...>
     *  Items: a|b|c Is a data packet for replacement
     *  example:{html|css|function}
     *
     *  Items: a:x:y
     *  x: name or Lambda expression
     *  y: Execution times
     *
     *  Lambda expression:
     *  Predefined Lambda is not implemented.
     *  If use Lambda then Lexical analysis ignores "y"
     *  Form: =>expr;
     *  expr: JavaScript expression.
     *  example:
     *        1.    =>packet[name][n]
     *        2.    =>Math.pow(2,n)
     *        3.    =>
     *
     * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * *
     *
     *  <s>
     *  Strategy: switch hex value. bit [00000111].
     *
     *  bit             description
     *  ---------------------------------------------------------------
     *  0.Replace       Default : true
     *                      If false: no replacement occurs
     *
     *  1.Sign          Default : true
     *                      Level:low
     *                      If n =  2 => start -> abc -> abc -> end
     *                      If n = -2 => start -> aa -> bb -> cc -> end
     *                      If false => n = |n|
     *
     *  2.Serial        Default : true
     *                      Level:high
     *                      If n = 2 => start -> abc -> abc -> end
     *                      If false => n = -|n| ;
     *                      serial_level > sign_level
     *
     *  3.Others        Unused
     *
